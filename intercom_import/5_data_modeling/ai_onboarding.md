# AI Onboarding

> Use Zoë (our AI analyst) to create your metrics for you

Select the tables you want to use for Zoë to create your metrics from the dropdown.

Zoë will look at your tables, create joins, and metrics that seem reasonable based on the columns present in the table. 

You can edit these definitions to update or change any of the initial values Zoë created for you.

You can select up to 6 tables at a time.
