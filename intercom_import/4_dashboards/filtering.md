# Filtering

> Add filters for slices and metrics to see exactly what you're looking for

Adding a filter

#### Add filter with left hand menu

You can use the left hand menu to filter on a metric of a slice by clicking the three dot menu and clicking "Add as filter". 

![Add+filter+sidebar](/assets/add+filter+sidebar.gif)

## Add filter with filter icon

You can also add a filter by clicking the filter icon below the question title.

![Filtericon](/assets/filtericon.gif)

#### Set the values for the filter

To set the value for the filter you can use the drop-down for the value on the right side of the value.

![Setvalues](/assets/setvalues.gif)
Edit filters

To edit a filter, click on the desired pill below the question title. 

#### Edit filter values

Click on the drop down to change the value of your chosen filter attribute.

![Editfiltervalues](/assets/editfiltervalues.gif)
#### Edit filter operators

You can also change the operators by clicking the middle drop-down.

![Comparison](/assets/comparison.gif)
#### Click 'Run Question' to apply filter changes

Make sure to click the 'Run Question' button once you add or change a filter to apply those changes to the question

![Hit+run+after+filter - gif)

#### OR Filters

You can also do OR filters and nested filters in Zenlytic. Just click "Add Filter" and then add filter rules or groups as needed to create the filter you're looking for.
