# Integrations Overview

Zenlytic integrates with the following data warehouses:

* Snowflake
  * Zenlytic pulls table and column level comments automatically on table import.
* BigQuery
  * Zenlytic pulls table and column level comments automatically on table import.
* Redshift
* Postgres
* MySQL
* Databricks SQL Warehouse
  * Zenlytic pulls table and column level comments automatically on table import.
* Druid
* DuckDB (MotherDuck)
* Trino
* SQL Server
* Azure Synapse Analytics
